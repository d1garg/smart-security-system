# 🔐 Smart Security System

A Python-based security project that captures a user's photo during login attempts using the webcam.

## 🚀 Features

* 📸 Automatically captures photo on login attempt
* ❌ Saves photo if password is incorrect
* ✅ Deletes photo if password is correct
* 🖥️ Simple GUI using Tkinter

## 🛠️ Tech Stack

* Python
* OpenCV (cv2)
* Tkinter
* OS & Time modules

## 📂 How It Works

1. User enters password
2. System captures photo instantly
3. If password is correct → photo is deleted
4. If password is wrong → photo is saved as evidence

## ▶️ How to Run

```bash
pip install opencv-python
python your_file_name.py
```
